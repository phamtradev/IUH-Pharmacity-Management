# Thư mục Hóa Đơn Bán Hàng

Thư mục này chứa các file PDF hóa đơn bán hàng được tự động tạo ra khi thực hiện giao dịch.

## Quy tắc đặt tên file

Format: `HoaDon_[MaDonHang]_[NgayThang].pdf`

Ví dụ: `HoaDon_DH001_27102025.pdf`

## Ghi chú

- File PDF tự động được tạo và lưu khi bấm nút "Bán hàng"
- Hỗ trợ tiếng Việt có dấu
- Có thể mở lại file PDF từ dialog preview hóa đơn
